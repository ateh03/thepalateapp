Palate App - University of South Carolina

Palate is the answer for any time you have a full fridge, but nothing to cook. Our solution connects ingredients you already have to recipes you'll love to try. Whether you're a seasoned chef, an aspiring cook, a foodie or someone looking for inspiration, Palate is your gateway to an endless culinary adventure. Learn more about the app on the Palate Wiki.

Palate is a native iOS / Android app written with React Native and Node.js using Firebase as a database for cloud and API data.

Setup

  • Download React Native

  • Download Node.js and npm

  • Download Android Studio

Clone the repository
Navigate to the Palate directory

You many have to add a local.properties file under the android directory and add the following line:
sdk.dir= <the path/location of your Android sdk>

Running

Open a terminal inside the directory

run npm install
run npm install @react-oauth/google@latest
run react-native start and then select android

Open Android Studio Code and run an AVD (Android Virtual Device)

Testing

Unit Testing
run npx jest
from the Palate directory The unit test is located in Palate/__tests/App.test.tsx and a mock of the firebase database as well as other configuration files are located in Palate/__mocks

Behavior Testing
located in teh Palate/e2e directory
(From the Palate Directory)
run npx run detox
run detox build --configuration android.emu.debug
